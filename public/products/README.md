## Notes

- **Hover photo is optional.** If you only have one photo per product,
  just copy it and name it both `[slug].jpg` and `[slug]-hover.jpg`.
  The hover effect still works — it will just show the same image.

- **No hover photo at all?** Delete the `-hover` filename entry and
  the site automatically falls back to the main photo.

- **Recommended specs:**
  - Size: 1000×1000px minimum (square crop)
  - Format: .jpg or .webp
  - Background: clean, neutral — ideally close to #F9EDEA (the site's petal tone)

- **While photos are missing**, the site shows beautiful gradient
  colour swatches as placeholders — so it looks great even before
  you add any images.
